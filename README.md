## 🎯 Описание проекта

Этот проект представляет собой портативный навигационный прибор на базе ESP32-C3 SuperMini с LED-матрицей 8×8. Устройство сочетает в себе:

- **Электронный компас** на основе магнитометра QMC5883L
- **GPS-навигатор** с отображением направления и расстояния до заданной точки
- **Индикатор заряда аккумулятора** с визуальной анимацией на матрице
- **Управление режимами** одной кнопкой

Идеально подходит для геокешинга и подобных квестов или просто как интересный DIY-проект.

---

## ⚡ Функционал

| Режим | Описание |
|-------|----------|
| **🧭 Компас** | Показывает направление на магнитный север. Стрелка на матрице указывает текущий азимут. Красная стрелка компаса |
| **🎯 GPS-направление** | При наличии GPS-фикса показывает направление (азимут) к сохранённой точке. Зеленая стрелка компаса |
| **📏 GPS-расстояние** | Отображает расстояние до сохранённой точки (в метрах) с цветовой индикацией. Если расстояние > 100 м, количество зеленых кваратиков - количество десятков метров. Если квадратиков матрицы не хватает и метров больше (7x7 * 10), каждый оранжевый квадратик означает 100 м. Если не хватает и оранжевых, то каждый красный означает 1км. |
| **🔋 Индикатор зарядки** | Автоматический режим при подключении зарядки: анимация заполнения батарейки. Зеленая батарейка при полной зарядке. Красная мигающая при проблемах с аккумулятором |

### Дополнительные возможности:
- **Регулировка яркости** LED-матрицы (удержание кнопки)
- **Сохранение GPS-точки** (тройной клик в GPS-режимах)
- **Калибровка компаса** (тройной клик в режиме компаса. Зеленая полоска при калибровке - процент сканирования пространства. Синяя - время калибровки)
- **Автосохранение настроек** в файловой системе LittleFS
- **Визуальная обратная связь** на LED-матрице

## 🎮 Управление

Управление осуществляется одной тактильной кнопкой:

| Действие | Результат |
|----------|-----------|
| **Удержание** | Регулировка яркости |
| **1 клик** | Переключение в режиме `GPS`: `GPS-направление` ↔ `GPS-расстояние` |
| **2 клика** | Переключение режима: `Компас` ↔ `GPS` |
| **3 клика** | В режиме `Компас` — **калибровка компаса**<br>В режиме `GPS` — **сохранение текущей позиции** |

> 💡 **Режим зарядки** активируется автоматически при подключении USB-питания. Кнопка в этом режиме неактивна.

## 🔌 Схема подключения

### Основные соединения

| ESP32-C3 | Подключение к |
|----------|---------------|
| **GPIO5** | Кнопка (GND при нажатии) |
| **GPIO0** | LED-матрица (DATA) |
| **GPIO8 (SDA)** | QMC5883L (SDA) |
| **GPIO9 (SCL)** | QMC5883L (SCL) |
| **GPIO20 (RX)** | GPS (TX) |
| **GPIO21 (TX)** | GPS (RX) — опционально |
| **GPIO1** | Датчик заряда (CHG) — LOW при зарядке. Берется с катода красного светодиода индикации зарядки с платы зарядки TP4056. Подключается через подтяжку через резистор 5-10кОм к 3.3V от платы ESP32  |
| **GPIO2** | Датчик заряда (FULL) — LOW при полной зарядке. Берется с катода синего светодиода индикации полного заряда с платы зарядки TP4056. Подключается через подтяжку через резистор 5-10кОм к 3.3V от платы ESP32 |

### Питание

Идеальная схема такая:
```
Внешний блок 5V 2A+
├── 5V  ├──► LED-матрица VCC
|       └──► ESP32-C3 5V
|
└── GND ├──► LED-матрица GND
        └──► ESP32-C3 GND
```

Но у нас:
```
Аккумулятор 3.7V 1A
├── 5V  ├──► LED-матрица VCC
|       └──► ESP32-C3 5V
|
└── GND ├──► LED-матрица GND
        └──► ESP32-C3 GND
```

> ⚠️ **Важно!** LED-матрица требует отдельного питания (не через ESP32). Ведите провод на матрицу сразу от аккумулятора. Общий GND обязателен для всех компонентов.

## 🚀 Установка и запуск (PlatformIO для новичков)

### 1. Установка PlatformIO

**Вариант A: Расширение для VS Code (рекомендуется)**

1. Скачайте и установите [Visual Studio Code](https://code.visualstudio.com/)
2. Откройте VS Code, перейдите во вкладку **Extensions** (Ctrl+Shift+X)
3. Найдите **PlatformIO IDE** и установите
4. Перезапустите VS Code

**Вариант B: Альтернативная IDE (CLion, Qt Creator)**

Для других IDE мы подготовили специальный файл `.idea` для быстрого запуска в CLion, а также расширение для Qt Creator. Вы можете использовать любой удобный редактор при наличии установленного PlatformIO Core.

### 2. Клонирование проекта

```bash
git clone https://github.com/SergTyapkin/Compass-Minecraft-ESP32.git
cd Compass-Minecraft-ESP3
```

**Если у вас нет Git:**
- Скачайте ZIP-архив проекта и распакуйте в папку

### 3. Открытие проекта

- В VS Code: **File → Open Folder...** → выберите папку проекта
- PlatformIO автоматически распознает проект

### 4. Подключение платы

1. Подключите ESP32-C3 к компьютеру через USB-C кабель
2. Убедитесь, что в системе появился COM-порт:
   - **Windows:** Диспетчер устройств → Порты (COM и LPT)
   - **Linux:** `ls /dev/ttyUSB*` или `ls /dev/ttyACM*`
   - **macOS:** `ls /dev/cu.usb*`

### 5. Сборка и загрузка прошивки

**Способ 1: Через интерфейс VS Code**
1. Нажмите на иконку PlatformIO в левой панели (в виде пиктограммы с головой робота)
2. В разделе **PROJECT TASKS** выберите:
   - **Build** — сборка проекта
   - **Upload** — загрузка прошивки на плату
   - **Upload and Monitor** — загрузка и открытие монитора порта

**Способ 2: Через терминал**
```bash
pio run --target upload --environment esp32-c3-super-mini
```

### 6. Мониторинг порта

Для просмотра отладочных сообщений:

```bash
pio device monitor --baud 115200
```

Или VS Code кнопка: **Upload and Monitor** сделает это автоматически.

### 7. Первый запуск

После успешной загрузки:
1. На LED-матрице должны загореться тестовые огни
2. В мониторе порта появится сообщение `✅ SETUP COMPLETE!`
3. Начните использовать устройство согласно инструкции управления
